Este post tiene contenido de relleno, con contenido básico de ejemplo usando el framwork de diseño web bootstrab, con detalles de su estructura, clases, estilos, contenidos, css y html5 de forma responsiva segun plantilla usada. Además contiene una Estuctura de diseño básico que se puede adaptar rapidamente a cualquier blog, sitio web etc..
Esta plantia y estuctura fue diseñada por mi persona, yasser lanuza como parte autoaprendizaje y test para futura implementación en proyectos, para diseño rapido, el contenido en esté artículo esta justicado con la página, lo que le da una vista ordenada.
